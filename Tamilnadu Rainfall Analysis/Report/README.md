# 🌧️ Tamil Nadu Rainfall Analysis (1901–2017)

## 📖 Project Overview

This project presents a statistical analysis of historical rainfall data for Tamil Nadu using Python. The analysis is based on rainfall records from the India Meteorological Department (IMD) covering the period **1901–2017**.

The project demonstrates the use of Python for statistical computation, data visualization, and trend analysis.

---

## 🎯 Objectives

- Analyze annual rainfall patterns in Tamil Nadu.
- Compute descriptive statistics.
- Analyze monthly and seasonal rainfall.
- Identify the wettest and driest years.
- Perform correlation analysis.
- Examine rainfall trends using Linear Regression.
- Validate trends using the Mann–Kendall Trend Test.
- Visualize rainfall anomalies.

## 📈 Key Findings

- Average annual rainfall: **940.47 mm**
- Wettest year: **2005 (1365.3 mm)**
- Driest year: **2002 (318.0 mm)**
- November showed the highest correlation with annual rainfall (**r = 0.585**).
- Linear regression indicated no statistically significant long-term rainfall trend.
- Mann–Kendall test confirmed the absence of a significant monotonic trend.
